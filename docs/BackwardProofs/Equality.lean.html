Reasoning about Equality

Equality (=) is also a basic logical constant. It is characterized by the following introduction and elimination rules:

Eq.refl : ‚àÄa, a = a
Eq.symm : ?a = ?b ‚Üí ?b = ?a
Eq.trans : ?a = ?b ‚Üí ?b = ?c ‚Üí ?a = ?c
Eq.subst : ?a = ?b ‚Üí ?p ?a ‚Üí ?p ?b

The first three lemmas are introduction rules specifying that = is an equivalence relation. The fourth lemma is an elimination rule that allows us to replace equals for equals in an arbitrary context, represented by the higher-order variable ?p.

The Eq.subst instance we use has ?a := a, ?b := a', and ?p := (Œª x => f a b = f x b):

a = a' ‚Üí (Œªx => f a b = f x b) a ‚Üí (Œªx => f a b = f x b) a'

In Œ≤-reduced form:

a = a' ‚Üí f a b = f a b ‚Üí f a b = f a' b

The lemma's first assumption matches the hypothesis ha : a = a', which is passed as an argument in the first apply invocation. The lemma's second assumption is a trivial equality that can be proved by apply Eq.refl or rfl. The lemma's conclusion matches the goal‚Äôs target. Notice how a higher-order variable (e.g., ?p) can represent an arbitrary context (e.g., f . . . b) around a term (e.g., a or a'). This works because apply unifies up to computation, including Œ≤-conversion.
